00030 /* Here is some guidance on logging levels to use:
00031  *
00032  * LOG_DEBUG    - For all messages that are enabled by optional debugging
00033  *        features, typically preceded by "if (IS...DEBUG...)"
00034  * LOG_INFO - Information that may be of interest, but everything seems
00035  *        to be working properly.
00036  * LOG_NOTICE   - Only for message pertaining to daemon startup or shutdown.
00037  * LOG_WARNING  - Warning conditions: unexpected events, but the daemon believes
00038  *        it can continue to operate correctly.
00039  * LOG_ERR  - Error situations indicating malfunctions.  Probably require
00040  *        attention.
00041  *
00042  * Note: LOG_CRIT, LOG_ALERT, and LOG_EMERG are currently not used anywhere,
00043  * please use LOG_ERR instead.
00044  */
